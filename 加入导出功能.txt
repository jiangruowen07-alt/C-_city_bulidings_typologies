================================================================================
加入导出功能 - 改动记录
================================================================================

一、概述
--------------------------------
在控制面板中新增导出控件，可将当前矩形城市场地导出为 PNG 图片。
采用程序化渲染（非屏幕截图），确保完整导出整个城市场地，不受 Windows 高 DPI 影响。


二、ui.py 修改
--------------------------------
1. 导入（约第 5-7 行）：
   - from tkinter import filedialog, messagebox
   - from PIL import Image, ImageDraw

2. build_layout 中，STEP/RESET/CLEAR 按钮下方（约第 145-148 行）：
   - 新增 exp_row 框架
   - 新增 "EXPORT PNG" 按钮，绑定 export_city_jpg 方法

3. 新增方法 export_city_jpg(self)（约第 747-802 行）：
   - 弹出保存对话框，默认文件名 city_export.png
   - 使用 PIL 程序化渲染城市场地：
     * 若为视图模式（PUBLIC/PRIVATE 或 CELL CONTRIBUTION）：绘制各格子颜色
     * 若为普通模式：绘制吸引子（矩形）和代理（椭圆）
   - 绘制网格线
   - 使用 max(self.cell, 8) 作为每格像素，保证导出清晰
   - 保存为 PNG 格式（无损）


三、requirements.txt
--------------------------------
新增依赖：
   Pillow>=10.0.0


四、使用说明
--------------------------------
1. 运行仿真（START SIMULATION 或 GO）
2. 在右侧控制面板点击 "EXPORT PNG"
3. 选择保存路径和文件名
4. 当前城市场地将导出为 PNG 图片


五、技术说明
--------------------------------
- 程序化渲染：根据模型数据直接绘制，不依赖屏幕截图
- 避免 DPI 问题：Windows 高 DPI 下，ImageGrab 与 tkinter 坐标可能不匹配
- 导出内容与当前视图一致（普通视图 / 公私有视图 / 贡献度视图）
