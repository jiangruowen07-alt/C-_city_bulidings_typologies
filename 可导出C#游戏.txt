--------------------------------------------------------------------------------
Grasshopper 使用说明 Usage in Grasshopper
--------------------------------------------------------------------------------

1. 在 Rhino 中打开 Grasshopper
2. 添加 C# Script 组件
3. 将导出的 .cs 文件内容粘贴进脚本
4. 输出端口：
   - A: Agent 位置 (List<Point3d>)
   - B: Agent 类型 (List<string>)
   - C: 总效用 (double)
   - D: 统计信息 (string)

5. 可选：在 C# 组件添加输入参数 (W, H, Steps, SwapMode)，取消脚本内对应注释以启用

--------------------------------------------------------------------------------
导出逻辑对应关系 Export Logic Mapping
--------------------------------------------------------------------------------

Python (model.py)          →  C# (export_csharp.py)
─────────────────────────────────────────────────
Agent(type, x, y)          →  Agent(Type, X, Y)
CityModel.get_utility()    →  CityModelGH.GetUtility()
CityModel.step()           →  CityModelGH.Step()
CityModel.calc_total_utility() → CityModelGH.CalcTotalUtility()
dist_attr (BFS)            →  DistAttr (BFS)
swap_mode                  →  SwapMode
pref_attr, pref_agent      →  PrefAttr, PrefAgent
reach_attr_by_type         →  ReachAttr
reach_agent_by_type        →  ReachAgent
